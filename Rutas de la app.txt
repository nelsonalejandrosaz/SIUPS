Rutas de la app

-- Rutas de usuarios -- 
/inicio/{rol} 			-> Inicio segun el rol

-- Gestion de usuarios -- 
/usuarios				-> Lista de usuarios en el sistema
/usuarios/{id}			-> Ver usuario por id
/usuarios/{id}/editar 	-> Editar usuario por id
/usuario/nuevo			-> Nuevo usuario
/usuario/{id}/eliminar	-> Eliminar usuario por id


-- Gestion de alumnos --
/alumnos				-> Lista de alumnos
/alumnos/{id}			-> Ver alumno por id
/alumno/{id}/editar		-> Editar alumno por id
/alumno/nuevo			-> Agregar alumno manualmente
/alumno/cargacsv		-> Carga de alumno por CSV

-- Exepciones de la app --
/permisodenegado		-> Permino denegado
